# Community Contributed Toolkits

Welcome to the Arcade AI community toolkits!

We're excited to see what you've built. Please submit your toolkit to the [community_toolkits.txt](community_toolkits.txt) file by opening a Pull Request with your toolkit added to the file.
When adding your toolkit, please follow the [CONTRIBUTING.md](CONTRIBUTING.md) file for guidelines on formatting your toolkit.

Once approved, your toolkit will be added to the [community toolkits page](https://docs.arcade-ai.com/integrations/community-toolkits) in our official documentation.

Documentation for your toolkit will be automatically generated and added to the [docs](https://docs.arcade-ai.com/integrations/community-toolkits).

If you would like to add additional documentation for your toolkit, we will use the README.md file located at the root of your toolkit package in addition to the automatically generated documentation.

Thank you for your contribution!
